# Institut-Transnational-de-Certification-des-Acteurs-Armés-ITCAA-
ITCAA est une plateforme transnationale de certification, de typification et de légitimation des fonctions armées non étatiques. Elle s’appuie sur l’intelligence artificielle, la cartographie géopolitique et les sciences informatiques pour structurer, reconnaître et transmettre les trajectoires des acteurs armées.
